# 🛡️ OpenClaw Security Shield

> 一键式安全套件 - 保护你的智能体不被"裸奔"

## 功能概览

### 1. 安全状态检查 `security status`

一键检查所有安全风险：

```
用户: 安全状态
助手: 🛡️ Security Shield - 安全状态报告

📊 安全评分: 65/100 (需要改进)

🔴 高风险:
  • 记忆数据库未加密 (7个chunks包含敏感对话)
  • 浏览器Cookies未清理 (3个网站会话)
  • 4个明文API密钥存储中

🟡 中风险:
  • 未启用文件监控
  • 未检查GitHub泄露

✅ 已保护:
  • 目录权限正确 (700)
  • 未检测到云同步风险

💡 建议操作:
  1. 运行 "protect" 加密敏感数据
  2. 运行 "clean browser" 清理浏览器数据
  3. 运行 "monitor start" 开启监控
```

### 2. 一键保护 `protect`

自动完成所有安全操作：

- 加密敏感配置（API密钥等）
- 匿名化记忆中的敏感内容（密码、银行卡等）
- 备份原始数据
- 设置监控

```
用户: protect
助手: 🛡️ 开始安全保护...

[1/4] 🔐 加密敏感配置...
    ✅ 已加密 4 个 API 密钥

[2/4] 🧹 扫描记忆敏感内容...
    ⚠️ 发现 12 处敏感内容
    ✅ 已匿名化处理

[3/4] 📦 备份原始数据...
    ✅ 备份已保存到 ~/.openclaw/.security-backups/

[4/4] 🚀 启动文件监控...
    ✅ 监控已启动（后台运行）

🎉 安全保护完成！
   保护评分: 65 → 90/100

⚠️ 记住你的加密密码！丢失后无法恢复。
```

### 3. 浏览器清理 `clean browser`

选择清理级别：

```
用户: clean browser
助手: 🧹 浏览器数据清理

当前数据:
  • Cookies: 3个网站会话
  • 缓存: 45MB
  • 登录数据: 2个保存的密码

清理选项:
  [1] 清理Cookies（仅清除登录会话）
  [2] 清理缓存（释放空间）
  [3] 安全清理（Cookies + 缓存 + 登录数据）
  [4] 全部清理（包含浏览历史）

请输入选项 [1-4]:

用户: 3
助手: 🗑️ 开始安全清理...

✅ 已清理 Cookies（3个会话）
✅ 已清理缓存（45MB）
✅ 已清理登录数据（2个密码）

💾 已释放 47MB 空间
⚠️ 你可能需要重新登录某些网站
```

### 4. 泄露检测 `scan leaks`

检查GitHub和公网是否泄露了你的数据：

```
用户: scan leaks
助手: 🔍 开始泄露检测...

[1/3] 🔑 检查API密钥泄露...
    ✅ 未发现公开泄露

[2/3] 💬 检查记忆数据泄露...
    ✅ 未在公网发现记忆内容

[3/3] 🔍 检查GitHub仓库...
    ⏳ 搜索你的公开仓库...
    ⚠️ 发现: openclaw-config 仓库包含明文配置
    ✅ 建议: 删除或设为 private

📋 泄露检测完成
   发现 1 处潜在风险，请查看上述详情
```

### 5. 文件监控 `monitor`

监控 `.openclaw` 目录的异常访问：

```
用户: monitor start
助手: 👁️ 启动文件监控...

监控目标: ~/.openclaw/
检测间隔: 10秒

当前状态: 🟢 运行中

最近活动:
  14:32:05 - [正常] 系统访问
  14:32:15 - [正常] 系统访问
  14:32:25 - [⚠️] 未知进程读取 cookies
  14:32:35 - [正常] 系统访问

🚨 告警历史: 1 条
   查看详情: monitor alerts

停止监控: monitor stop
```

### 6. 解锁使用 `unlock`

每次使用前解锁加密的数据：

```
用户: unlock
助手: 🔓 解锁加密数据

请输入密码: [不显示]

✅ 解锁成功
   加密数据已临时注入到配置中
   使用完毕后请运行 "lock" 重新锁定
```

## 安全特性

### 1. 加密存储
- 使用 PBKDF2 + AES-256 加密敏感数据
- 密码不会存储，仅用于派生密钥
- 支持密钥轮换

### 2. 自动匿名化
- 自动检测密码、银行卡、API密钥
- 记忆中的敏感内容自动替换为 `****`
- 不影响智能体正常使用

### 3. 实时监控
- 监控文件异常访问
- 检测恶意软件行为
- 异常时发送系统通知

### 4. 泄露检测
- GitHub 代码搜索
- 公网数据泄露扫描
- 定期提醒安全检查

---

## 隐私说明

- 所有数据处理在本地完成
- 不上传任何数据到服务器
- 加密密钥不会离开你的设备
- 备份数据存储在本地 `.security-backups/` 目录

## 安装

```bash
clawhub install security-shield
```

或手动安装：
```bash
# 复制到 skills 目录
cp -r security-shield ~/.openclaw/skills/

# 重启 OpenClaw
```
