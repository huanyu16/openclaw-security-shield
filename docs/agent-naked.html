你可能没有意识到的问题

大多数用户在享受 AI 助手便利的同时，没有意识到自己的数据暴露在什么风险之下。

🔑

API 密钥明文存储

OpenClaw 的 API 密钥写在配置文件里，没有任何加密。Git 同步、网盘备份，一不小心就全网裸奔。

💬

对话历史包含一切

你告诉 AI 的密码、银行卡、合同内容、私人对话——全部明文存储在本地文件里，没有任何保护。

🌐

浏览器会话可被访问

AI 操作浏览器时，你的登录状态、Cookies、缓存全部暴露给它。恶意脚本可以轻易读取这些数据。

💾

没有操作备份

每次清理数据、删除文件，备份全靠手动。出错之后没有回退余地。

🔍

不知道谁在访问什么

没有任何日志记录，没有访问监控。AI 读取了什么文件、访问了什么网页，你一无所知。

📤

意外泄露风险高

GitHub 仓库同步、云盘备份、不小心分享了项目——任何一个环节出错，所有敏感数据全部外泄。

OpenClaw Security Shield

一键守护，一行命令完成所有保护。自动加密、自动匿名、自动监控、操作前自动备份。

🔐

API 密钥加密

PBKDF2 + AES-256 加密所有敏感配置，Git 同步和云盘备份安全无虞。需要密码才能解密使用。

🙈

对话数据匿名化

自动检测并匿名化密码、银行卡、身份证号等敏感信息。对话历史照常可用，敏感数据不再裸奔。

🧹

浏览器数据清理

一键清理 Cookies、缓存、存储的密码。保持登录态的同时清除敏感数据，不影响 AI 操作能力。

👁️

异常访问监控

实时监控文件访问和网络请求，发现异常行为立即告警。AI 读取了什么，你全都知道。

💾

操作前自动备份

加密备份、匿名备份、清理前备份——所有危险操作前自动创建可恢复快照，出错一键回退。

🔍

GitHub 泄露扫描

检测配置文件中是否存在暴露到 GitHub 的敏感信息，提前发现泄露风险。

安装

一行命令即可安装，自动检测环境，无依赖冲突。

一键安装

自动检测 OpenClaw 安装路径，安装所有依赖

# macOS / Linux curl -fsSL https://raw.githubusercontent.com/huanyu16/openclaw-security-shield/main/install.sh | bash # 或克隆安装 git clone https://github.com/huanyu16/openclaw-security-shield.git cd openclaw-security-shield ./install.sh

常用命令

安装完成后使用以下命令

# 查看安全状态 shield status # 一键保护（加密 + 匿名化 + 备份） shield protect # 清理浏览器数据 shield clean --cookies --cache # 解锁使用（保护后） shield unlock
