OpenHanako 的优势
以人为本 · 交互深度
  • 🧠衰减式记忆 — 模拟人类记忆的时间衰减，近期清晰远期淡出
  • 💫意识模板 — Agent 真正有"性格"，人格是 Markdown 文件，备份即拷贝
  • 🛡️原生双层沙箱 — 不依赖 Docker，用 OS 级隔离，对普通用户更友好
  • 📁书桌空间 — 人机异步协作的独特交互模式
  • 📔日记 + 经验提取 — Agent 能自我反思和持续学习
  • 📦零配置开箱即用 — 下载安装即可，不需要命令行或 Docker
OpenClaw 的优势
以功能为本 · 覆盖广度
  • 🌐全渠道覆盖 — 20+ 消息平台适配，远超竞品
  • 📱原生 App — iOS/Android/macOS 原生应用，体验更轻量
  • 🔌Plugin SDK — 成熟的插件开发接口和扩展系统
  • 🏗️企业级工程 — 20K commits、完整 CI/CD、多层测试矩阵
  • 🔊语音 + 画布 — 内置 TTS 语音合成和实时 Canvas 渲染
  • 🌍完整国际化 — i18n 模块支持多语言

打个比方：如果 AI 助手是一部手机——

OpenClaw 像 Android：开放、可定制、适配所有场景，功能覆盖极广。
OpenHanako 像 iPhone：一体化、精致、开箱即用，在交互深度上打磨到极致。

两者不完全是竞品，更像不同的产品哲学——一个追求覆盖广度，一个追求交互深度。✨
